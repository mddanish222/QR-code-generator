# MD-QR Studio

MD-QR Studio is a modern Progressive Web App (PWA) for generating and scanning QR codes.

## Features

- Generate QR codes for:
  - Website URLs
  - Plain Text
  - Phone Numbers
  - SMS
  - Email
  - vCard Contacts
  - Wi-Fi Login
  - WhatsApp Messages
  - UPI Payments
  - Google Maps
  - Social Media Links
  - PDF Links
  - Voting QR Codes

- QR customization:
  - Multiple QR styles
  - Custom QR size
  - Add center logo
  - Frame styles
  - Custom labels
  - Download QR as PNG

- QR Scanner:
  - Live camera scanner
  - Upload image scanner
  - Flip camera support

- Extra:
  - QR history
  - WhatsApp sharing
  - Copy QR data
  - Install as PWA app

## Tech Stack

- HTML
- CSS
- JavaScript
- Service Worker
- Web App Manifest
- QR Code Styling Library
- jsQR
